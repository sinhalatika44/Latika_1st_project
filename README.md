# Programming basics

- variables: `let`, `const`, `var`
- operators: `+`, `-`, `*`, `/`, `%`
- datatypes: `string`, `number`, `boolean`, `object`
- comparison operators: `===`, `==`, `!==`, `!=`
- conditional operator: `if else`, `if - else if - else`
- AND and OR operator: `&&`, `||`
- Objects: JSON `{name: 'Latika', age: 23, gender: 'Female'}`

- Loops: `for loop`

### function example

```
function sum(a, b) {
  const c = a + b;
  return c;
}

const num = sum(2, 4);
console.log(num);
```
